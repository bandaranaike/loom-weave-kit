# @erbitron/loom-weave-kit

Core browser-safe loom design toolkit.

## Features

- Product/fabric specifications
- Palette + mixed color recipes
- Layered design model
- Deterministic compiler to row-RLE
- Constraint validation engine
- JSON, SVG, and weaver-sheet exports

## Usage

```ts
import { createDesign, createPalette, compileDesign } from '@erbitron/loom-weave-kit';
```

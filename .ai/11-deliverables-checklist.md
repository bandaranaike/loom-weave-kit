# 11 Deliverables Checklist

Mark complete only when all are true.

1. Monorepo tree exists with required files and folders.
2. Core source files are implemented (not placeholder stubs for compile/constraints/export).
3. Package build scripts and export maps are configured.
4. Tests pass.
5. Root and package READMEs exist and are aligned with features.
6. Example usage snippets/files are present.
7. Core imports in browser builds without Node polyfills.
8. Example runs produce:
- compiled RLE
- constraint report
- SVG preview
- weaver sheet text
9. Canvas package produces PNG in a browser.
10. React hooks support compile and undo/redo without UI components.
